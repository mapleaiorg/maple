# Resonator Layer (`crates/resonator/*`)

Resonator is the cognition/lifecycle layer for MAPLE entities.

Components:

- `types`: Core Resonator cognitive/state types.
- `identity`: Identity and continuity primitives.
- `meaning`: Meaning formation abstractions.
- `intent`: Intent stabilization abstractions.
- `commitment`: Commitment-related abstractions.
- `profiles`: Profile constraints and profile contracts.
- `runtime`: Runtime coordination for Resonator flows.
- `client`: Client-facing helpers.
