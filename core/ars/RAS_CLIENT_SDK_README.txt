ARS client SDK implementation. The SDK provides:

1. ARSClient: Unified client supporting both REST and gRPC
2. ARSRestClient: REST-specific client implementation
3. ARSGrpcClient: gRPC-specific client implementation
4. ClientConfig: Comprehensive configuration options
5. AgentManager: High-level agent lifecycle management
6. BatchOperations: Helper for bulk operations
7. MockARSClient: Testing utilities
8. Automatic protocol selection and failover
9. Built-in caching for improved performance
10. Event streaming support for real-time updates

The SDK is designed to be easy to use while providing advanced features for complex scenarios.
